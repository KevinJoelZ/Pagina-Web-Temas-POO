Temas de POO

1. Clases

Una clase es una plantilla o modelo para crear objetos. Define las propiedades y m√©todos que tendr√°n los objetos.

  • Definici√≥n: Con la palabra clave class
  • Atributos: Variables que describen el objeto

2. Objetos

Un objeto es una instancia concreta de una clase que tiene estado (atributos) y comportamiento (m√©todos).

  • Estado: Valores de sus atributos

El constructor es un m√©todo especial que se ejecuta al crear un objeto. Se usa para inicializar los atributos.

  • Nombre: Igual al nombre de la clase
  • Se ejecuta: Al usar new
  • Tipos: Por defecto, parametrizado, sobrecargado
  • this: Referencia al objeto actual

4. Herencia

La herencia permite crear nuevas clases basadas en clases existentes, reutilizando c√≥digo y estableciendo relaciones "es-un".

  • Clase padre: Superclase o clase base
  • Clase hija: Subclase que hereda

  • Ventaja: Herencia m√∫ltiple

6. Clases Abstractas

Una clase abstracta es una clase que no puede ser instanciada directamente. Se usa como base para otras clases.

  • Palabra clave: abstract
  • M√©todos abstractos: Sin cuerpo, solo firma
  • Herencia: Se extiende con extends
  • Utilidad: Definir estructuras comunes

El polimorfismo permite que objetos de diferentes clases sean tratados de manera uniforme a trav√©s de una interfaz o clase com√∫n.

  • Tipos: Sobrecarga y sobreescritura
  • Sobre carga: Mismo m√©todo, diferentes par√°metros
  • Sobre escritura: Redefinir m√©todo de padre
  • Upcasting: Tratar subclase como superclase
